🔑 Riot API Key

This app is 100% static — it runs entirely in your browser and needs your own free Riot API key. Get one in 30 seconds at developer.riotgames.com (log in → copy the "Development API Key"). The key is stored only in your browser (localStorage) and sent only to Riot's servers. Dev keys expire every 24h — for a permanent key, register a free "Personal" app on the same page.

☁️ GitHub sync (optional)

Paste a GitHub token with the gist scope (create one here) and your analyzed match data is saved to a private Gist automatically after every analysis — and restored from it on any device, instead of relying on this browser's localStorage.

📁 JSON file backup

Prefer a plain file? Export your analyzed data as JSON, and import it later (or on another machine).

⚔️ LoL HardCounter

Enter your Riot ID (name#tag), pick your region and hit Analyze.

It studies your match history and tells you what to pick against whom — based on your own winrates, not just global stats. Plus builds, runes, live-game scouting and team-comp insights.
